# 발표 전날, 마지막으로 이것만 챙깁시다 (Cookeat)

기능 범위와 서버 검증은 이미 충실합니다. 오늘은 첫인상과 위생을 다듬는 데 시간을 쓰면 좋겠습니다.

- 레시피 상세·목록에 회색 플레이스홀더 이미지가 아직 그대로 노출됩니다. 실제 음식 사진으로 채우는 게 완성도를 가장 크게 끌어올리는 작업이라, 여유가 되면 이걸 먼저 해주세요.

- 새로 붙인 문의 컴포넌트 쪽에서 lint 경고가 몇 건 유입됐습니다. 마지막으로 한 번 정리하고, 가능하면 CI에서 lint가 실제로 검사되도록 걸어두면 깔끔합니다.

- 판매자 대시보드·통계·문의관리를 발표에서 보여줄 텐데, 화면에 데이터가 그럴듯하게 차 있어야 살아 보입니다. 시드 데이터를 미리 채워두고, 사용자 → 판매자 → 관리자 흐름을 순서대로 리허설해 주세요.
