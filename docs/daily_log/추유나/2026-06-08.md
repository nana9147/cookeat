- **작업**: `BasicInfoField.tsx`에 판매 상태 추가

  - `ProductStatus` 타입 활용

### 5. ProductFormData nested 구조 리팩토링

- **작업**: `types/seller/product.ts`, `index.tsx` 리팩토링
- **내용**:
  - `ProductFormData`를 `basicInfo`, `pricingInfo`, `shippingInfo`, `productInfo`, `returnPolicy` nested 구조로 변경
  - `index.tsx` 제네릭 핸들러 `handleChange` 하나로 통합
  - 각 컴포넌트 `data`, `onChange` props 방식으로 통일
  - `BasicInfoData`, `PricingData`, `ProductInfoData`, `ReturnPolicyData`, `FoodType` 타입 추가

  - 식품 유형은 기존 `CategoryName` 타입 활용 + `'기타'` 추가 (`FoodType`)
