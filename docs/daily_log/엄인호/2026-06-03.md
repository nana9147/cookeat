## 세부 작업

### 1. Cookeat 디자인 시스템 CSS 정의

- `app/globals.css`에 색상 토큰, 타이포그래피 스케일 정의
- `@theme`으로 커스텀 브레이크포인트 설정 (mobile: 390px, tablet: 768px, desktop: 1440px)
- `@theme inline`으로 CSS 변수를 Tailwind 토큰에 매핑
- `.vscode/settings.json`에 CSS 린트 경고 무시 설정 추가

### 2. 헤더 컴포넌트 구현

- `components/Header/Header.tsx` — 헤더 진입점, 서브 컴포넌트 조립
- `components/Header/HeaderNav.tsx` — 데스크탑 네비게이션 링크 (홈/레시피/재료 쇼핑)
- `components/Header/HeaderSearch.tsx` — 데스크탑 검색 인풋
- `components/Header/HeaderActions.tsx` — 로그인 버튼(비로그인) / 유저·장바구니 아이콘(로그인), 햄버거 버튼
- `components/Header/HeaderIcons.tsx` — SVG 아이콘 컴포넌트 모음
- `store/authStore.ts` — Zustand 인증 토큰 상태
- `store/header/headerStore.ts` — Zustand 사이드바 열림/닫힘 상태
- `app/layout.tsx` — 루트 레이아웃에 Header 추가

### 3. Cookeat 로고 및 파비콘 추가

- `app/favicon.ico` 교체

- `components/Header/SidebarUser.tsx` — 비로그인: 로그인 버튼 / 로그인: 유저·장바구니 아이콘
