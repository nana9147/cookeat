### 3. Supabase 연동 및 인증 상태 관리

- `lib/supabase.ts` — 클라이언트 전용 Supabase 인스턴스 (publishable key)
- `lib/supabaseAdmin.ts` — 서버 전용 Supabase 인스턴스 (service role key, API 라우트 전용)
- `lib/api.ts` — Axios 인스턴스 (Bearer 토큰 자동 첨부·공통 에러 처리)
- `services/auth/authService.ts` — 인증 API 호출 서비스 레이어
- `services/auth/authTypes.ts` — AuthUser·LoginResult·RegisterResult 타입 정의
- `store/authStore.ts` — Zustand persist 스토어에 hydration 완료 플래그 추가
- `store/headerStore.ts` — 헤더 UI 스토어 (서브폴더 제거)
- `components/(auth)/AuthInitializer.tsx` — 앱 시작 시 Supabase 세션 복원·토큰 자동 갱신
- `app/(auth)/layout.tsx` — 로그인 상태이면 홈으로 자동 리다이렉트
- `app/layout.tsx` — AuthInitializer 추가
