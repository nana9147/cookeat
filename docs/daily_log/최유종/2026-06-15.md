## 세부 작업

### 1. 관리자 회원관리 페이지 DB 연동 및 API 라우트 추가

- `app/api/admin/users/route.ts` (신규)
  - `requireAdmin`으로 관리자 인증 가드 적용
  - `keyword`(닉네임·이메일 ilike 검색), `status`, `page`, `limit` 쿼리 파라미터 지원
  - Supabase `supabaseAdmin`으로 `users` 테이블 조회, `orders(count)`로 주문 수 집계
  - `orderCount >= 10` 조건으로 서버에서 `grade` 계산 후 응답
- `app/api/admin/users/[userId]/status/route.ts` (신규)
  - `PATCH` 핸들러로 회원 상태(`active` / `suspended`) 변경
- `app/admin/members/page.tsx`
  - 하드코딩 목 데이터 제거, `/admin/users` API 실데이터 연동
  - 상태 필터·페이지 파라미터를 API에 전달하도록 변경
  - 회원 상태 수정 다이얼로그에서 `PATCH /admin/users/:id/status` 호출
  - `Member` 인터페이스, `statusLabel` 매핑 정의
